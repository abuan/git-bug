Les commandes décrites dans ce dossier correspondent à des commandes liées à l'interface du logiciel de base git_bug, elles ont été déliées de la commande principale.
Elles permettent de créer un bridge vers des dépots type Gitlab/github / lancer un terminal adapté au modèle d'objet Bug / lancer une interface Web dédiée au modèle Bug
Elles peuvent être supprimées, si besoin.
